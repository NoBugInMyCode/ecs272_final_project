## How to Run
1. Download dataset from [here](https://www.kaggle.com/datasets/fronkongames/steam-games-dataset)
2. Unzip the .json file into `/data_preprocessing`
3. run `python data_preprocessing/make_slim_json.py`
4. `cd frontend`
5. `npm start` (make sure React is installed)
